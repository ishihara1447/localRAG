第三者ソフトウェア・モデルのライセンス全文を同梱している。

## 注意事項

- `llama3.1:8b` は Phase 1 検証用の暫定 LLM であり、Meta の
  Llama 3.1 Community License Agreement (MIT/Apache とは異なる
  独自ライセンス) が適用される。月間アクティブユーザー数 7 億人
  超の場合は別途 Meta からライセンスを取得する必要があるなど、
  MIT/Apache-2.0 系とは異なる利用条件が課されるため、本番配布前に
  同ライセンスの内容を必ず確認すること。
- 本番想定の `llm-jp/llm-jp-4-8b-thinking` に切り替えた場合は
  Apache-2.0 のみとなり、上記の追加条件は課されない
  (`CLAUDE.md` の技術スタック方針を参照)。
- `trust_remote_code` を要するモデルはコードの事前レビューと
  コミットハッシュ固定が必須（`CLAUDE.md` 「絶対に守るルール」参照）。
- モデルファイル自体（重み）はこのリポジトリに含まれない。
  `scripts/export.sh` が生成する配布パッケージの `ollama-models/`
  配下に実体が含まれる。
- WinSW / Node.js は Windows native 配布版
  (`windows-native/export-windows.ps1` が生成するパッケージ) にのみ
  同梱される。
- `Node.js_LICENSE.txt` は Node.js 本体 (MIT) に加え、同梱される
  第三者コンポーネント (ICU, OpenSSL, V8, zlib, npm 等) のライセンス
  全文を含む複合ライセンスファイルである。正は実際に同梱する Node
  配布物 (node-v22.x-win-x64) 内の LICENSE ファイルであり、同梱する
  Node のバージョンを固定・変更した際は配布物内の LICENSE と本ファイル
  の内容が一致していることを確認すること。
