- カテゴリ: c
  質問: 「非核三原則」とは何ですか。

  根拠: 「非核三原則とは、 核兵器を持たず、 作らず、 持ち込ませずという原則を指し、 わが国は国是としてこれを堅持している。」 ===PAGE 206===

- カテゴリ: c
  質問: 白書のいう「反撃能力」とはどのような能力ですか。
  正解: 武力攻撃が発生し「武力の行使」の三要件を満たす場合に、必要最小限度の自衛の措置として相手の領域で有効な反撃を加える、スタンド・オフ防衛能力などを活用した自衛隊の能力。
  採点パターン: キーワード=［三要件／相手の領域／スタンド・オフ防衛能力／必要最小限（度）］
  根拠: 「反撃能力とは、 わが国に対する武力攻撃が発生し…やむを得ない必要最小限度の自衛の措置として、 相手の領域において、 わが国が有効な反撃を加えることを可能とする、 スタンド・オフ防衛能力などを活用した自衛隊の能力をいう。」 ===PAGE 213===

- カテゴリ: c
  質問: 「文民統制（シビリアン・コントロール）」とはどのような考え方ですか。
  正解: 民主主義国家における軍事に対する政治の優先、または軍事力に対する民主主義的な政治による統制。
  採点パターン: キーワード=［軍事に対する政治の優先／民主主義／統制］
  根拠: 「文民統制は、 シビリアン・コントロールともいい、 民主主義国家における軍事に対する政治の優先、 または軍事力に対する民主主義的な政治による統制を指す。」 ===PAGE 206===

- カテゴリ: c
  質問: 「衛星コンステレーション」とはどのようなシステムですか。
  正解: 一定の軌道上に多数の小型人工衛星を連携させて一体的に運用するシステムで、衛星通信や情報収集などに使われる。
  採点パターン: キーワード=［多数の小型人工衛星／軌道／一体的に運用／情報収集］
  根拠: 「一定の軌道上に多数の小型人工衛星を連携させて一体的に運用するシステムのことであり、衛星通信や画像などの情報収集などの用途で使われる。」 ===PAGE 228===

- カテゴリ: c
  質問: 憲法第9条第2項でいう「交戦権」とは何を指しますか。
  正解: 戦いを交える権利という意味ではなく、交戦国が国際法上有する種々の権利の総称。
  採点パターン: キーワード=［交戦国／国際法上有する種々の権利／総称］
  根拠: 「ここでいう交戦権とは、 戦いを交える権利という意味ではなく、 交戦国が国際法上有する種々の権利の総称であって、 相手国兵力の殺傷と破壊、 相手国の領土の占領などを行うことを含むものである。」 ===PAGE 205===
