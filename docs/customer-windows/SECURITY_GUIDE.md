# LocalRAG セキュリティガイド（Windows 版）

- 対象バージョン: v1.0.0
- 作成日: 2026-07-10

このガイドは、「アップロードした文書や質問内容が外部に送信されない」
という LocalRAG の最重要方針について、その根拠を説明するものです。
顧問先への説明や所内のセキュリティ確認にもお使いいただけます。

## 1. 基本方針

LocalRAG は、お客様の文書・質問・回答・チャット履歴を
**インターネット上のサービス（クラウド AI 等）へ一切送信しない**
ことを前提に設計されています。AI モデル本体を含むすべての部品が
お使いのパソコンの中で動作し、インストールから日常利用まで
インターネット接続を必要としません。

## 2. データが外部送信されない根拠

### 2-1. AI の処理がすべてパソコン内で完結する

- 回答を生成する AI（LLM: `llm-jp-4-8b-thinking`、日本語特化モデル）と、
  文書を検索用データに変換する AI（embedding: `mxbai-embed-large`）は、
  どちらも**配布パッケージに同梱**され、このパソコンのディスク
  （`C:\ProgramData\LocalRAG\models`）から読み込まれます。
  利用時にインターネットからダウンロードすることはありません。
- これらのモデルを動かす実行エンジン（Ollama）は、このパソコンの
  内部だけで通じるアドレス **`127.0.0.1:11435`** で動作します。
  `127.0.0.1` は「このパソコン自身」を意味する特別なアドレスで、
  他のパソコンやインターネットからは到達できません。
  すべての AI 推論はこの内部通信のみで行われます。

### 2-2. 外部 AI サービスは仕組みとして遮断されている

OpenAI・Anthropic・Google Gemini などの外部 AI サービスへの接続機能は、
画面から選べないようにしてあるだけでなく、**プログラム内部（API
レベル）で許可リスト方式により遮断**しています。設定を操作しても
外部 AI プロバイダを有効化できない作りになっています。

### 2-3. その他の外部通信機能も既定で無効

| 機能 | 状態 | 説明 |
|---|---|---|
| テレメトリ（利用状況の開発元への送信） | 無効 | 利用状況・統計情報が外部に送信されることはありません |
| Web スクレイピング（Web ページの取り込み） | 無効 | 本ビルドでは機能自体を同梱していません。取り込み対象はローカルのファイルのみです |
| Swagger API ドキュメント（開発者向け API 仕様画面） | 無効 | 攻撃の足がかりになる情報公開を減らすためです |
| モデル配布サイト（Hugging Face）への実行時アクセス | 禁止（オフラインモード） | モデルやコードが動作中に外部から取得されることはありません |

これらの設定は配布時点で固定されており、変更しないでください。

### 2-4. お客様側でも確認できます

LocalRAG はネットワークがなくても動作するため、
**LAN ケーブルを抜く／Wi-Fi を切った状態でも、文書のアップロードと
質問応答が通常どおり動作すること**をお客様自身で確認できます。
外部サービスに依存していれば、この状態では動作しません。

## 3. データの保存場所（すべてこのパソコン内）

| データ | 保存先 |
|---|---|
| アップロードした文書・抽出テキスト・検索用データ・チャット履歴・設定 | `C:\LocalRAG\app\server\storage`（インストール先を変更した場合はその配下） |
| AI モデル本体 | `C:\ProgramData\LocalRAG\models` |
| 動作記録（ログ） | `C:\ProgramData\LocalRAG\logs` |
| バックアップ | `C:\ProgramData\LocalRAG\backups`（既定） |

クラウドストレージや外部サーバーには保存されません。したがって、
これらのフォルダおよびバックアップ zip の管理（アクセス権、
バックアップ媒体の保管）は、お客様の所内の情報管理ルールに
従って行ってください。

## 4. 利用にあたってのお願い

- **Windows のログインパスワードと LocalRAG の管理者パスワード**は、
  所内の規程に従って適切に管理してください。データはこのパソコン内に
  あるため、パソコン自体の管理が最も重要です。
- **ディスクの暗号化（BitLocker 等）**の利用を推奨します。パソコンの
  盗難・廃棄時の情報漏えい対策になります。
- **API キー**（プログラムから LocalRAG を操作するための鍵。
  Settings → API Keys で発行）を発行した場合は、平文でのメール送付や
  共有ドキュメントへの貼り付けを避け、不要になったら無効化して
  ください。
- バックアップ zip には文書・履歴がそのまま含まれます。外部媒体へ
  コピーする際は、機密文書と同じ扱いで保管してください。
- 本製品のバージョンを勝手に更新・変更しないでください。同梱
  ソフトウェアはすべてバージョンを固定して検証済みの組み合わせで
  配布しています（`versions.lock` に記録されています）。

## 5. ライセンス

同梱している OSS（AnythingLLM、Ollama、WinSW、Node.js）および
AI モデル（llm-jp-4-8b-thinking、mxbai-embed-large）のライセンス・
著作権表示は、インストール先の `LICENSES\` フォルダと `NOTICE`
ファイルに同梱しています。これらの表示を削除しないでください。
